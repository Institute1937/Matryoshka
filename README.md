Matryoshka - CoreOS Vagrant DevBox
---------------

* Runs docker inside vagrant
* Guest port 80 maps to host port 2600
* Mounts . to /usr/core/institute1937


TLDR
----------------
```
git clone git@github.com:Institute1937/Matryoshka.git
cd Matryoshka
vagrant up
vagrant ssh
```
